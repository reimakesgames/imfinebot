# Access Codes

Internally, there are types that represent what commands are available to a user.
These are access codes and are represented by the `AccessCode` enum.

The access codes are as follows:

| Access Code | Description |
| ----------- | ----------- |
| 0 | No access. |
| 1 | General user access. |
| 2 | Server administrator access. |
| 3 | Server owner access. |
| 4 | Bot staff access. |
| 5 | Bot developer access. |
| 6 | Bot owner access. |
